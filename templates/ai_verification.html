How to Verify AI is Actually Working

1. Check Token Usage

Real AI calls consume tokens. Check the token_usage table:

SELECT * FROM token_usage ORDER BY date DESC;

If tokens increase, AI is making real calls

2. Test Student Chat

Login as student, ask a subject-specific question. Check:

  • Response should be relevant
  • Should refuse off-topic questions
  • Response saved in chat_message table
3. Generate Quiz

Use quiz generation feature. Verify:

  • Questions are unique (AI-generated)
  • Match the subject and difficulty
  • Different each time (not templates)
4. Check SQLite Database

The three agents use school_ai.db. To verify:

sqlite3 school_ai.db
SELECT * FROM student_profiles;
SELECT * FROM quiz_history;
